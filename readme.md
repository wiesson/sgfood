### Install

`go get github.com/wiesson/sgfood`

### Run

`~ ❯❯❯ sgfood`

```
🐟  Fisch mit Kräuterkruste auf Parmesan Polenta und Kohlrabi Ragout
🌸  Tofu mit Kräuterkruste auf Parmesan Polenta und Kohlrabi Ragout
```

#### Options

`~ ❯❯❯ sgfood -help`

#### To lazy to read? 

Pipe results to TTS (OSX) and listen.

`~ ❯❯❯ sgfood | say -v Anna`
